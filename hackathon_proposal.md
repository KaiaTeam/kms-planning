# Kaia Native USDT 머니 스트리밍 DApp

## Kaia 블록체인 기반 실시간 결제 혁신

> **Korea Stablecoin Hackathon 2025**  
> 매초마다 흐르는 돈, 언제든 출금하는 자유

---

## 프로젝트 개요

### 핵심 아이디어
**"일한 순간부터 돈이 쌓이는" 실시간 결제 시스템**

지금 이 순간에도 전 세계 수백만 명의 프리랜서들이 프로젝트를 완료할 때까지 돈 한 푼 받지 못하고 있습니다. 직장인들은 월말까지 기다려야만 급여를 받을 수 있습니다. 크리에이터들은 플랫폼의 월말 정산을 기다리며 현금 부족에 시달립니다.

우리는 이 "기다림"을 없애고 싶습니다. 일한 순간부터 매초마다 돈이 실시간으로 적립되어, 언제든 원하는 만큼 출금할 수 있는 시스템을 만들었습니다.

### 해결하는 핵심 문제들
1. **"언제 돈을 받을지 모르는" 불안감 해소**: 매초 쌓이는 금액을 실시간 확인
2. **"돈이 없어서 못하는 일들" 제거**: 언제든 일한 만큼 바로 출금 가능
3. **"월말까지 기다리는" 스트레스 제거**: 하루 일한 만큼 그날 받기 가능
4. **"플랫폼에 의존하는" 불안정성 탈피**: 직접적이고 투명한 거래

## 왜 이 서비스가 필요한가?

### 실제 비즈니스 문제

**프리랜서: "프로젝트는 끝났는데 돈은 언제 들어와?"**
- 3개월 열심히 일했지만 완료까지 생활비 0원
- 대출받아서 버티거나 다른 일을 또 해야하는 현실
- 좋은 기회가 와도 "돈이 없어서" 포기하는 상황 반복
- 핵심 문제: **일과 돈의 시간차**

**직장인: "월급날만 기다리는 인생"**
- 매일 열심히 일해도 월말까지 통장은 텅텅
- 아이가 갑자기 아파도 "월급날까지 기다리라고" 할 수 없음
- 좋은 기회나 할인 혜택을 놓치는 경우가 반복됨
- 핵심 문제: **노동과 보상의 시간차**

**크리에이터: "내가 번 돈인데 언제 받을 수 있을까?"**
- 열심히 방송해도 언제 얼마나 받을지 알 수 없음
- 갑작스런 장비 고장이나 이사 등에 대처할 수 없음
- 플랫폼 정책 변화에 속수무책으로 당할 수밖에 없음
- 핵심 문제: **창작과 수익의 시간차**

### 우리가 제공하는 해결책

**"일한 만큼 바로 받는다"는 당연한 권리 실현**
- 1시간 일하면 1시간치 돈이 바로 쌓임
- 급한 일이 생기면 지금 당장 출금해서 해결
- 더 이상 "언제 받을지 모르는" 불안함 없음

**실제 변화하는 일상**
- 월요일에 일한 돈으로 화요일에 맛있는 걸 먹을 수 있음
- 아이가 아프면 바로 병원비를 낼 수 있음
- 좋은 기회나 할인이 있으면 즉시 참여할 수 있음
- "돈이 없어서"라는 말을 더 이상 하지 않아도 됨

**삶의 질 변화**
- 스트레스: "언제 받을까?" → "얼마나 쌓였을까?"
- 계획: "월급날까지 버티자" → "내일 뭐 할까?"
- 자유: "돈 때문에 못해" → "하고 싶은 걸 하자"

## 문제 해결 과정: 어떻게 쓰나요?

### 1단계: 복잡한 절차 없이 바로 시작
**해결하는 문제: "절차가 너무 복잡해서 포기하게 되는 것들"**
- **기존 문제**: 새로운 결제 방식 도입하려면 몇 주씩 걸림
- **우리 해결책**: 지갑 연결 30초면 끝, 바로 사용 시작
- **실제 변화**: "나중에 하자"가 "지금 해보자"로 바뀜

### 2단계: 내 돈 상황 한눈에 파악
**해결하는 문제: "내가 얼마나 벌었는지, 언제 받을지 모르겠어"**

**이제 볼 수 있는 것들**
- **지금 이 순간 출금 가능한 돈**: 더 이상 "언제 받을지" 고민 안 해도 됨
- **매초 쌓이는 내 돈**: 일하는 보람이 숫자로 실시간 확인
- **투명한 계약 현황**: 누가 나에게 얼마를 언제까지 보내는지 명확히

**달라지는 경험**
- **주는 입장**: "직원이 언제까지 버틸까" 걱정 → "매일 조금씩 받으니 안정적이겠네"
- **받는 입장**: "언제 입금될까" 불안 → "지금 얼마 쌓였나" 설렘
- **모든 거래**: 불투명함 → 실시간 투명함

### 3단계: 공정한 계약 맺기
**해결하는 문제: "언제 얼마를 받을지 애매모호한 약속들"**

**이제는 명확하게**
- **누구에게**: 정확한 사람
- **얼마를**: 정확한 금액
- **언제부터 언제까지**: 정확한 기간
- **얼마씩**: 매초 얼마씩 쌓일지도 자동 계산

**달라지는 계약 관계**
- **고용주**: "급여 줄 때마다 눈치 안 봐도 되네"
- **직원**: "매일 조금씩 받으니까 회사 상황이 어려워도 불안하지 않네"
- **클라이언트**: "중간중간 주니까 프리랜서가 더 열심히 하네"
- **프리랜서**: "완료까지 기다리지 않아도 되니까 집중할 수 있네"

### 4단계: 실시간으로 진행 상황 확인
**해결하는 문제: "지금 상황이 어떻게 되고 있는지 모르겠어"**

**이제 실시간으로 보이는 것들**
- **"지금까지 얼마 받았나?"**: 숫자가 실시간으로 올라감
- **"앞으로 얼마나 더 받을까?"**: 남은 기간과 금액 명확히
- **"언제 끝나나?"**: 정확한 종료일과 현재 진행률
- **"문제없이 잘 되고 있나?"**: 상태를 실시간으로 확인

**상황에 따른 유연한 대처**
- **일이 잘 안 될 때**: 송금자가 일시정지 가능
- **다시 괜찮아질 때**: 언제든 재개 가능
- **아예 그만둘 때**: 공정하게 정산하고 종료
- **모든 결정**: 양쪽 다 투명하게 확인 가능

## 실제 사용 시나리오

### 시나리오 1: 프리랜서 김개발자의 하루

**기존: "프로젝트 끝날 때까지 버텨야지"**
```
1월: 프로젝트 시작, 통장 잔고 50만원
2월: 열심히 개발 중, 부모님께 용돈 부탁
3월: 프로젝트 완료, 그동안 못한 일들 산더미
4월: 드디어 900만원 입금, "이제 살았다"
```

**이제: "일한 만큼 바로바로 받으니까"**
```
1월 1일: 프로젝트 시작, 돈도 바로 쌓이기 시작
1월 8일: "어? 벌써 21만원 쌓였네" → 맛있는 거 먹기
1월 31일: "300만원 쌓였으니" → 새 노트북 사기
매일매일: 일하는 재미가 달라짐, 성취감 업
```

김개발자는 이제 일 자체에 집중할 수 있고, 돈 걱정 없이 더 좋은 결과를 만들어냅니다.

### 시나리오 2: 스타트업 직원 박과장의 급여

**기존: "월급날까지 어떻게 버티지?"**
```
3월 1일: 새 달 시작, 지난달 다 써서 잔고 부족
3월 15일: 아이 갑자기 아픔, "월말까지 기다려" 할 수 없음
3월 25일: 카드값 결제일, 연체료 걱정으로 밤잠 설침
3월 31일: 급여 입금, "이번 달도 간신히 버텼다"
```

**이제: "매일 일한 만큼 받으니까"**
```
3월 1일: 일 시작하자마자 급여도 쌓이기 시작
3월 15일: "150만원 쌓였으니" → 병원비 바로 해결
3월 25일: "250만원 있으니" → 카드값도 여유롭게
매일 퇴근길: "오늘도 10만원 쌓였구나" → 뿌듯함
```

박과장은 이제 돈 걱정 없이 일과 가정에 집중할 수 있습니다.

### 시나리오 3: 크리에이터 이유튜버의 후원

**기존: "이번 달 후원금은 언제 들어올까?"**
```
매일 열심히 방송, 시청자들도 후원 많이 해줌
→ 하지만 플랫폼에서 "월말에 정산해서 줄게"
→ 장비 고장나도 "정산 때까지 기다리자"
→ "이번 달은 얼마나 벌었을까" 궁금해도 정확히 모름
```

**이제: "실시간으로 후원받으니까"**
```
방송 중에 후원 오면 바로바로 내 계정에 쌓임
→ "어? 오늘 벌써 33만원 쌓였네" 실시간 확인
→ 장비 고장나면 "지금 있는 돈으로" 바로 해결
→ 시청자들도 "바로 전달된다니" 후원 의욕 업
```

이유튜버는 이제 안정적인 수입으로 더 좋은 콘텐츠를 만들 수 있습니다.

## 데모 시연 계획

### 1단계: 문제 인식 (2분)
"여러분, 프리랜서로 일해보신 분 계신가요? 3개월 프로젝트 하면서 완료할 때까지 돈 한 푼 못 받는 경험, 있으시죠?"

현재 결제 시스템의 문제점을 실제 경험과 함께 설명하겠습니다.

### 2단계: 솔루션 소개 (2분)  
"이제 매초마다 일한 만큼 돈이 쌓이는 시대가 왔습니다."

화면에 실시간으로 숫자가 올라가는 모습을 보여드리겠습니다.

### 3단계: 라이브 데모 (6분)

**3-1. 지갑 연결 (30초)**
- kaia 지갑 연결하는 모습
- 복잡한 가입 절차 없이 바로 사용 가능함을 보여줌

**3-2. 마이페이지 둘러보기 (1분)**  
- 깔끔한 인터페이스
- 실시간 잔액 표시
- 계약 목록들이 한눈에 보이는 모습

**3-3. 새로운 급여 계약 만들기 (2분)**
- "김개발자에게 월 300만원 급여" 스트림 생성
- 자동으로 계산되는 초당 금액 (1.16원/초)
- 스마트 컨트랙트 서명 과정

**3-4. 실시간 모니터링 (2분)**
- 계약 상세 페이지에서 숫자가 실시간으로 변하는 모습
- "현재 전송된 금액"이 매초마다 1.16원씩 증가
- 김개발자 입장에서 언제든지 출금할 수 있음을 강조

**3-5. 계약 관리 (30초)**
- 필요시 계약을 중도에 취소할 수 있는 기능
- 투명하고 공정한 거래 관리

### 4단계: 기술적 우수성 (2분)
- 3개 테이블만으로 구현한 효율적 설계
- 지갑 기반 인증의 보안성과 편의성
- Kaia 블록체인의 빠른 속도와 낮은 수수료

### 5단계: 실제로 달라지는 삶들 (2분)
- **프리랜서**: "돈 때문에 좋은 프로젝트 포기" → "원하는 일에 집중"
- **직장인**: "월급날까지 버티는 삶" → "매일이 월급날인 삶"
- **사업주**: "직원들 눈치 보며 급여" → "자동으로 공정하게"
- **창작자**: "플랫폼 눈치 보는 창작" → "팬들과 직접 소통하는 창작"

## 왜 이 프로젝트가 성공할 것인가?

### 진짜 문제 해결
- **"돈이 없어서" 포기했던 일들**: 이제 할 수 있음
- **"언제 받을지 몰라" 생기는 스트레스**: 이제 없음
- **"월말까지 버텨야" 하는 답답함**: 이제 자유로움

### 우리가 만들고 싶은 세상  
- **일하면 바로 받는** 당연한 세상
- **"돈이 없어서"라는 말이 사라진** 세상
- **일한 만큼 받는 게 투명하고 공정한** 세상

### 달라지는 일상의 경험
- **일할 때**: "오늘도 돈이 쌓이고 있구나" 뿌듯함
- **계획할 때**: "돈 모을 때까지 기다리자" → "지금 있는 돈으로 하자"
- **관계에서**: "돈 빌려달라" → "내일 출금해서 갚을게"

### 적용 분야 확장
- **개인간 결제**: 프리랜서 → 직장인 → 모든 일하는 사람
- **비즈니스 결제**: 급여 → 임대료 → 구독료 → 모든 정기지급
- **국제 결제**: 국경 없는 실시간 결제로 글로벌 비즈니스 가능

## 마무리

**"매초마다 흐르는 돈, 언제든 출금하는 자유"**

우리가 만든 것은 단순한 기술이 아닙니다. 일하는 모든 사람들의 **"기다림"**을 없애는 혁신입니다.

복잡한 금융상품이나 어려운 블록체인 기술이 아닌, **"일하면 바로 받는다"**는 당연한 권리를 되찾아주는 서비스입니다.

더 이상 **"돈이 없어서"** 포기하지 마세요. 더 이상 **"언제 받을지 몰라서"** 불안해하지 마세요. 더 이상 **"월말까지"** 기다리지 마세요.

**이것이 바로 기술이 사람의 삶을 더 자유롭게 만드는 진정한 혁신입니다.**

감사합니다.
